¿Qué significa CSS?

Significa "cascading style sheet"

¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

Sirve para personalizar una pagina web, y se necesita un navegador web para interpretar su codigo

¿A qué se llama regla de CSS?

la regla del css es un "selector" que se encarga de definir a que instancia se aplican los estilos, una "propiedad" que es lo que permite dar un estilo
un valor de propiedad y estas dos ultimas cosas hacen la declaracion

¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

Selector
selecciona el(los) elemento(s) a dar estilo. Para dar estilo a un elemento diferente, solo cambia el selector.
Declaracion
aplica las propiedades que querés dar al elemento seleccionado en el Selector. En este caso hay una sola regla, pero puede haber más.
Propiedades
es lo que te permite dar un estilo a un elemento HTML. En el ejemplo la propiedad es color que le dará color a las letras de los párrafos. Siempre las propiedades se ubicarán a la izquierda.
Valor de propiedad
las propiedades deben tener un valor. Cada propiedad tiene valores definidos. Cada valor se ubicará a la derecha.

¿Qué es un selector en CSS?

Es la primer parte de una regla css, es basicamente lo que le indica al navegador que cosa se esta modificando

Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

Los selectores de id son "#", los de clases, son ".", los de etiquetas son la etiqueta sin nada adelante, el universal es "*", y el descendente es "reverse"

Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

  1. El navegador carga el HTML
  2. Despues convierte el HTML en un DOM (modelo de objetos del documento)
  3. Entonces el navegador busca todos los elementos vinculados al HTML como los videos incrustados o las imagenes ademas del CSS vinculado
  4. El navegador analiza el CSS y ordena en diferentes cubos las diferentes reglas segun el tipo de selector
  5. El arbol de renderizacion muestra la estructura en que los nodos deberian aparecer despues de aplicar las reglas
  6. En la pantalla se muestra el aspecto visual de la pagina (esta etapa se llama pintura)
